- PostgreSQL

<h2>How to Run</h2>

- clone the project

```bash
git clone https://github.com/abhay-707/focus-hub.git
```

- install backend dependancies

```bash
cd backend
npm install
```

- install frontend dependancies

```bash
cd frontend
npm install
```

- setup .env files
- start servers

```bash
cd backend
npm run dev
```

```bash
cd frontend
npm run dev
```
